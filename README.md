# Nota
Open source Google keep clone built with Vue and Firebase mainly for practice.

### Features
- Create, read, edit and delete notes
- Filter by tag or color
- Search notes
- User authentication

### Tools
- Vue (router, vuex, pwa)
- Firebase (Authntication, Firestore)
